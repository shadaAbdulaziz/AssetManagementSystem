General Concept:
This application is an asset management system designed to help organize and track assets across different offices. 
It allows users to add, delete, view, and manage assets efficiently while supporting multiple currencies and locations.
The system calculates asset lifecycle information, such as end-of-life dates and status (new, used, or old), and provides
sorting features for better data organization. 
It also includes a user-friendly interface for easy interaction and management of assets.
